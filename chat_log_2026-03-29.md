## 四、Pilot Crew 後端架構設計

### 核心概念

後端 Multi-Agent 依功能包裝為 6 個 **Pilot Crew**，由 **Co-Pilot 指揮艙**（workflowService）統一編排。

```
Co-Pilot 指揮艙（MAF 工作流引擎）
    ↓ 依貸款類型召集對應 Crew
房貸：文件 → 法規 → 徵信 → 鑑估 → 審議 → 報告
信貸：文件 → 法規 → 徵信 → 審議 → 報告
```

### 六大 Pilot Crew

| Crew | 英文名 | 核心職責 | 適用 |
|------|--------|---------|------|
| Crew 1 | Credit Intelligence Pilot Crew | 5P 評級 + DBR/負債比 + 8 項防詐 | 房貸 ✅ 信貸 ✅ |
| Crew 2 | Valuation Pilot Crew | XGBoost 估價 + Monte Carlo 信心區間 | 房貸 ✅ 信貸 ❌ |
| Crew 3 | Regulatory Pilot Crew | RAG 三大知識庫問答 + 合規檢查 | 房貸 ✅ 信貸 ✅ |
| Crew 4 | Committee Pilot Crew | 三位 AI 委員審議 + 最終決策 | 房貸 ✅ 信貸 ✅ |
| Crew 5 | Document Pilot Crew | 謄本解析 + 徵審報告 + PDF 批覆書 | 房貸 ✅ 信貸 ✅ |
| Crew 6 | Customer Service Pilot Crew | LINE Bot + LIFF + 進度推播 | 房貸 ✅ 信貸 ✅ |

### 統一 Crew Response 格式

```typescript
interface PilotCrewResponse {
  crewId: string;
  crewName: string;
  status: "success" | "partial" | "failed";
  duration: number;
  confidence: number;
  result: object;
  recommendation: string;
  warnings: string[];
}
```
