- ✅ **三條核心指令**:

- ✅ **上下文監控**: 使用 `/showline` 監控上下文百分比（<50% 需重開）

### 技巧 3️⃣: 使用自分形 & 自描述

- ✅ **自動更新**: 源碼變動時逐級更新註釋和 CLAUDE.md

### 技巧 4️⃣: 清晰的提示詞

### 技巧 6️⃣: Agent 協作

## 🎯 專案特定規則

### 前端對話流程（技巧 4️⃣ 清晰提示詞）
1. **貸款類型判斷優先**：首先詢問房貸 or 信貸
2. **基本資訊收集**：年齡、職業、收入、用途、年限、金額
3. **房貸專屬資訊**：屋齡、坪數、車位、格局、樓層、建物類型
4. **智能產品推薦**：
   - 房貸：青安🥇 → 國軍🥈 → 一般🥉
   - 信貸：國軍🥇 → 薪轉🥈 → 一般🥉
5. **視覺海報生成**：70%主產品 + 30%交叉銷售
6. **月付金計算**：
   ```
   月付金 = 本金 × (月利率 × (1+月利率)^期數) ÷ ((1+月利率)^期數 - 1)
   ```

### 後端徵審模型（雙模式）
#### 信貸模式（DBR 22倍）
```
DBR = 無擔保債務總額 ÷ 月收入
必須 ≤ 22倍（金管會規定）

無擔保債務包含：
- 信用卡循環
- 現金卡
- 信用貸款
- 其他無擔保債務
```

#### 房貸模式（負債比）
```
負債比 = (月還款總額 + 18,000) ÷ 月收入

月還款總額包含：
- 所有金融機構房貸月付
- 所有金融機構信貸月付
- 信用卡最低應繳
- 其他貸款月付

最低生活費：18,000 元/月（全國統一）

評等標準：
- ≤ 70%：優良 ✅
- 70-85%：尚可 🟡
- > 85%：建議婉拒 🔴
```

### 5P 評級系統
```
每項 1-10 級評分：
1. 職業穩定性（People）
2. 所得成長性（People）
3. 資金用途合理性（Purpose）
4. 資產淨值（Payment）
5. 淨值比率（Payment）
6. 流動比率（Payment）

綜合評分：加權平均後 0-100 分
```

### 防詐檢測（8項）
```
需至少通過 6 項：
1. ✅ 證件真實性（OCR + MyData）
2. ✅ 公司居住地檢核
3. ✅ 聯徵查詢次數（2月≤3次）
4. ✅ 人頭戶風險評估
5. ✅ 收入來源合理性
6. ✅ 職業金額匹配度
7. ✅ 聯絡電話真實性
8. ✅ 異常申貸行為

風險等級：
- 通過 7-8 項：低風險 🟢
- 通過 6 項：中風險 🟡
- 通過 <6 項：高風險 🔴
```

### ML 鑑價模型（僅房貸）
```
模型組合：
1. LSTM 時間序列（9個月歷史）
   - 輸入：房價指數 + 利率
   - 輸出：下月房價指數預測
   - 誤差：< 0.6%（論文驗證）

2. Random Forest + SDE（市場情緒）
   - 長期參數：30個月
   - 短期參數：10個月
   - Monte Carlo：1,000 路徑模擬
   - 準確率：93.35%

3. 風險評估：
   - 情緒 << 0：看跌（調整 0.95）
   - 情緒 ≈ 0：中性（調整 1.0）
   - 情緒 >> 0：看漲（調整 0.97）

學術基礎：王文楷 (2023) 住宅學報
```

### RAG 法規問答（三大知識庫）
```
優先權層級：
1. 🔴 央行不動產抵押貸款問答（最高）
   - 用途：LTV、寬限期、高價房
   
2. 🟡 個金政策性貸款規章（中）
   - 用途：新青安、國軍貸款規定
   
3. 🟢 個金授信規章（一般）
   - 用途：徵信、保證人、一般流程

技術：
- Azure AI Search（語義搜尋）
- GPT-4o（答案生成 + 精確引用）
- Cosmos DB（多輪對話記憶）
- Hot Cache（常見問題 <1秒）
```

/**
 * INPUT: LoanApplication（貸款申請數據）, JCICData（聯徵數據）
 * OUTPUT: DebtRatioResult（負債比計算結果含明細）
 * POS: 後端徵審服務核心計算模組，為房貸審核提供負債比評估依據
 */

import { LoanApplication, JCICData, DebtRatioResult } from '../models';

export class DebtRatioCalculator {
  /**
   * 計算房貸負債比
   * 公式：(月還款總額 + 18,000) ÷ 月收入
   */
  calculate(application: LoanApplication, jcicData: JCICData): DebtRatioResult {
    const monthlyPayments = this.calculateTotalMonthlyPayments(jcicData);
    const minimumLivingCost = 18000; // 全國統一最低生活費
    const monthlyIncome = application.customerInfo.monthlyIncome;
    
    const debtRatio = (monthlyPayments + minimumLivingCost) / monthlyIncome;
    
    return {
      debtRatio,
      monthlyPayments,
      minimumLivingCost,
      monthlyIncome,
      grade: this.determineGrade(debtRatio),
      pass: debtRatio <= 0.85
    };
  }
  
  private calculateTotalMonthlyPayments(jcicData: JCICData): number {
    let total = 0;
    
    // 房貸月付
    jcicData.housingLoans?.forEach(loan => {
      total += loan.monthlyPayment;
    });
    
    // 信貸月付
    jcicData.creditLoans?.forEach(loan => {
      total += loan.monthlyPayment;
    });
    
    // 信用卡最低應繳
    total += jcicData.creditCardMinPayment || 0;
    
    // 其他貸款
    jcicData.otherLoans?.forEach(loan => {
      total += loan.monthlyPayment;
    });
    
    return total;
  }
  
  private determineGrade(debtRatio: number): '優良' | '尚可' | '建議婉拒' {
    if (debtRatio <= 0.7) return '優良';
    if (debtRatio <= 0.85) return '尚可';
    return '建議婉拒';
  }
}
