## 九、Teams Adaptive Card 格式問題修正

### 問題

完整版 Adaptive Card（含 Container / FactSet / Separator）發送後 HTTP 202 但 Teams 未顯示新訊息。
Power Automate Workflows webhook 對複雜元素支援有限。

### 解決

將卡片格式降版並統一改用 TextBlock：

| 變更項目 | 舊版 | 新版 |
|---------|------|------|
| Adaptive Card 版本 | `1.4` | `1.2` |
| 標題列 | Container（style: attention）| TextBlock（color: Attention）|
| 案件資訊 | FactSet | 多行 TextBlock |
| 分隔線 | Separator | 移除 |

**修改檔案**：`src/main/typescript/services/powerAutomateNotifier.ts`（57 行 → 16 行，精簡 -70%）
