## 三、P2-B 不動產市場資訊主動推播

### 設計目標

維持客戶黏著度：讓未進行中申請的客戶每週收到有價值的市場資訊，適時喚起購屋/貸款需求。

### 架構

```
客戶輸入「訂閱市場資訊」
→ subscribeUser(userId) 寫入 data/market-subscribers.json
→ 回覆訂閱成功確認卡（首刊日期 + 內容預告）

每週一 09:00（台灣時間）
→ startMarketPushScheduler（啟動時計算至下一個週一的 timeout）
→ pushWeeklyMarket()
→ 4 週輪播市場資料集（利率走勢 + 均價漲跌 + 熱門區域 + 庫存天數）
→ buildMarketInfoFlex → push 給所有訂閱者
→ Footer：「立即試算房貸」+ 「取消訂閱」
```

### 市場資料集（4 週循環）

| 週 | 利率 | 均價 | 趨勢 | 熱門區域 |
|----|------|------|------|---------|

| B | 2.06% ─ | 42.3萬/坪 ▼ | 跌 -0.5% | 中正、松山 |
