### 2-2 對話狀態機流程

#### 房貸流程（12 個資料收集步驟）

```
CHOOSE_LOAN_TYPE
→ COLLECT_AGE（年齡）
→ COLLECT_OCCUPATION（職業，Quick Reply 6 選項）
→ COLLECT_INCOME（月收入）
→ COLLECT_PURPOSE（貸款用途）
→ COLLECT_TERM（年限）
→ COLLECT_AMOUNT（金額）
→ COLLECT_EMPLOYER（任職公司）
→ COLLECT_PROPERTY_AGE（屋齡）
→ COLLECT_AREA（坪數）
→ COLLECT_PARKING（車位）
→ COLLECT_LAYOUT（格局）
→ COLLECT_FLOOR（樓層）
→ COLLECT_BUILDING_TYPE（建物類型）
→ RECOMMEND
```

#### 信貸流程（7 個資料收集步驟）

```
CHOOSE_LOAN_TYPE
→ COLLECT_AGE → COLLECT_OCCUPATION → COLLECT_INCOME
→ COLLECT_PURPOSE → COLLECT_TERM → COLLECT_AMOUNT
→ COLLECT_EMPLOYER
→ RECOMMEND
```

#### 以房養老獨立流程（RA Flow，60歲以上客群）

```
「以房養老」（全域觸發）
→ RA_INTRO（暖金色入口介紹卡）
→ RA_COLLECT_AGE（60–85 歲驗證）
→ RA_COLLECT_AREA（Quick Reply：20坪以下 / 20–40 / 40–60 / 60坪以上）
→ RA_COLLECT_PROP_AGE（Quick Reply：10年以內 / 10–20 / 20–30 / 30年以上）
→ RA_MONTHLY_WISH（Quick Reply：3萬 / 5萬 / 8萬 / 讓銀行算）
→ RECOMMEND（loanType = REVERSE_ANNUITY）
```

UX 設計原則：≤ 4 個 Quick Reply、不收職業/收入、語氣親切緩慢。

#### AI 建議模式

```
AI_SUGGEST_Q1（主要需求：購屋/週轉/消費）
→ AI_SUGGEST_Q2（有無房屋，週轉情境用）
→ RECOMMEND
```

#### 文件備妥與申請流程

```
RECOMMEND
→ CONFIRM_APPLY（LIFF 電子申請書）
→ PREPARE_DOCS（文件說明）
→ CONFIRM_MYDATA（MYDATA 所得資料）
→ CONFIRM_LAND_REG（土地建物謄本，房貸 / 以房養老）
→ COLLECT_NAME → COLLECT_PHONE
→ APPLY_DONE（顯示案件編號）
→ UPLOAD_DOCS（圖片上傳）
→ DOC_REVIEW（文件資料確認）
```

### 2-5 訊息類型處理

| 訊息類型 | 觸發條件 | 處理方式 |
|---------|---------|---------|
| 文字 | 任意狀態 | 全域指令攔截 → 狀態機轉移 |

#### 台語語音辨識流程

```
LINE 音訊訊息
→ 下載 base64
→ Breeze-ASR-26（或 Demo fallback：3 組示範台語句）
→ parseVoiceWithClaude NLU
  （解析：loanType / income / amount / purpose / termYears / occupation）
→ 填入 session
→ hasEnough → RECOMMEND / 不足 → CHOOSE_LOAN_TYPE
→ buildVoiceResultFlex 確認卡（辨識原文 + 解析欄位 + 確認/修改按鈕）
```

## 三、P2-B 不動產市場資訊主動推播

### 設計目標

維持客戶黏著度：讓未進行中申請的客戶每週收到有價值的市場資訊，適時喚起購屋/貸款需求。

### 架構

```
客戶輸入「訂閱市場資訊」
→ subscribeUser(userId) 寫入 data/market-subscribers.json
→ 回覆訂閱成功確認卡（首刊日期 + 內容預告）

每週一 09:00（台灣時間）
→ startMarketPushScheduler（啟動時計算至下一個週一的 timeout）
→ pushWeeklyMarket()
→ 4 週輪播市場資料集（利率走勢 + 均價漲跌 + 熱門區域 + 庫存天數）
→ buildMarketInfoFlex → push 給所有訂閱者
→ Footer：「立即試算房貸」+ 「取消訂閱」
```

### 市場資料集（4 週循環）

| 週 | 利率 | 均價 | 趨勢 | 熱門區域 |
|----|------|------|------|---------|
| A | 2.06% ▲ | 42.5萬/坪 ▼ | 跌 -0.3% | 大安、信義 |
| B | 2.06% ─ | 42.3萬/坪 ▼ | 跌 -0.5% | 中正、松山 |
| C | 2.07% ▲ | 42.8萬/坪 ▲ | 漲 +1.2% | 新店、板橋 |
| D | 2.05% ▼ | 42.1萬/坪 ─ | 持平 | 內湖、南港 |

### 新增檔案

| 檔案 | 說明 |
|------|------|
| `config/marketSubscriberStore.ts` | JSON 訂閱名單（subscribe/unsubscribe/isSubscribed） |
| `services/marketPushService.ts` | 市場資料、Flex 建構、pushWeeklyMarket、排程器 |
| `api/marketPush.ts` | Admin API（trigger/preview/stats） |

### Admin API

| 端點 | 說明 |
|------|------|
| `POST /api/admin/market-push/trigger` | 立即觸發推播（測試用） |
| `GET /api/admin/market-push/preview` | 預覽本週 Flex JSON（不推播） |
| `GET /api/admin/market-push/stats` | 查看訂閱人數 |
