    → Azure AI Search Hybrid Search 向量搜尋法規三庫
    → Azure OpenAI GPT-4o 合成法規答案 + 自動生成批覆書草稿
    → 行員確認即可，不需從頭撰寫

  Crew 4 防詐審議（異常偵測 + Critic Agent）

    → 委員會決議為建議，行員做最終 HITL 決策

  Crew 5 行銷客服（推薦系統 + ASR）
    → 三層推薦引擎：Content-Based 資格篩選 → CF 套餐組合 → LLM 個人化說明

       • 延遲固定：不受雲端負載影響，不存在「降智」問題
       • 資料主權：推論在 Azure Container Apps 容器內，客戶資料不出租戶
       • 繁體中文 + 金融術語優化：DBR / 成數 / 寬限期理解更自然

       ↓

       ↓

底部說明：

投影片 5.5【前端體驗革命：LINE Bot × 智慧推薦】
標題：從導流工具到服務平台：Co-Pilot LINE Bot 的前端體驗重塑
副標：無斷點申辦 × 即時推薦 × 台語語音 — 讓每個客戶都有專屬數位理專

左欄【現行合庫 LINE Bot 的問題】（SmartArt Before）：
❌ 功能極少，基本上只能查詢和導流
❌ 點任何功能 → 強制彈出視窗
❌ 跳出 LINE → 導入官網或網銀 App
❌ 流程斷裂，客戶中途放棄率高
❌ 無法在 LINE 內完成申辦，只是入口
❌ 沒有推薦邏輯，所有客戶看到一樣的內容

右欄【Co-Pilot LINE Bot 的突破】（SmartArt After）：
✅ 全程在 LINE 內完成，零跳出、零彈窗
✅ 對話式收集需求，自然語言輸入
✅ 台語語音輸入：「我想借錢買厝」直接辨識
✅ 即時推薦：填完基本資料立刻顯示最適合產品
✅ 個人化推薦說明：同一產品對不同客戶說不同理由
✅ 申辦狀態即時推播：「您的案件已進入審議階段」
✅ 不打擾：只在關鍵節點推播，不騷擾

中間對比數字（SmartArt 大字）：
現行體驗：點擊 → 彈窗 → 跳 App → 填表 → 放棄
Co-Pilot：對話 → 推薦 → 確認 → 送件（全程 LINE 內，< 3 分鐘）

下半部【推薦系統三層引擎說明】：

Layer 1 資格篩選（Content-Based）：
  客戶屬性（年齡 / 職業 / 收入 / 用途）→ isEligible() 自動過濾不符資格產品
  金融優勢：必填欄位 = 完整 User Profile，冷啟動問題天然不存在

Layer 2 套餐推薦（Collaborative Filtering）：
  「申辦青安的客戶，68% 同時辦了御璽卡 + 房貸壽險」
  生命週期套餐自動組合：

投影片 6【推薦系統：演算法架構與學術依據】
標題：三層推薦引擎的技術底層：從資格篩選到 LLM 指令遵循
副標：Content-Based × Collaborative Filtering × InstructRec 2023

SmartArt 三層堆疊（由下往上）：

Layer 1【Content-Based Filtering — 資格篩選層】：
  輸入：客戶屬性（年齡 / 職業 / 收入 / 貸款用途）
  機制：isEligible() 硬規則過濾 → 候選商品池
  設計原則：推薦的產品一定能申辦，零客訴風險
  對比 Netflix：Netflix 靠行為猜偏好，我們靠填表得到完整 User Profile

Layer 2【Collaborative Filtering — 套餐推薦層】：
  機制：同溫層行為模式 → 生命週期金融組合
  輸出：主推產品（70% 版面）+ 交叉銷售（30% 版面）
  套餐示範：
    首購族（30歲/公務員）→ 青安房貸 + 御璽卡 + 房貸壽險
    退休族（62歲/自有房）→ 以房養老 + 安養信託 + 黃金存摺
    軍公教（職業軍人）→ 國軍輔導房貸 + 軍人認同卡 + 團保
  可解釋性：「和您條件相似的客戶，有 68% 同時申辦了這個組合」

Layer 3【LLM Instruction Following — 個人化說明層】：
  機制：申請表單 = Structured Instruction（客戶填表即下指令）
  執行：Qwen2.5:14b 開源模型，本地穩定推論，不受雲端降智影響
  亮點：同一產品，護理師看到「月省 X 元」，軍人看到「軍人專屬優惠」
  學術依據：Recommendation as Instruction Following（ArXiv 2023，InstructRec）

底部對比說明：
「傳統規則引擎：所有軍人看到一樣的推薦，說一樣的話。
 Co-Pilot 三層引擎：精準推薦 + 個人化說明 + 生命週期套餐，說對的話給對的人。」
