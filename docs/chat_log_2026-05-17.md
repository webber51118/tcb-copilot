## 五、SHAP 實裝紀錄

### 背景
PPT 大綱寫有「SHAP 前三大貢獻因子」，但實際上 SHAP 只在防詐（CREW 3）有實裝，鑑估（CREW 2）沒有。

### 實裝步驟

**Step 1｜Python：`xgboostValuationService.py`**
- 加入 `_explainer = None`（全域）
- `_load()` 後嘗試建立 `shap.TreeExplainer`
- `valuate_xgboost()` XGBoost 模式：呼叫 `_shap_factors_live(row)` 計算真實 SHAP
- Demo 模式 fallback：`_shap_factors_demo()` 規則近似（屋齡/樓層/行政區）
- 回傳新增 `shap_factors: [{label, contribution, direction}]`
- 安裝：`pip3 install shap`（v0.49.1）

**Step 2｜TypeScript types：`models/types.ts`**
- 新增 `ShapFactor` interface
- `XGBoostValuationResult` 加 `shapFactors: ShapFactor[]`
- `ValuationResult` 加 `shapFactors?: ShapFactor[]`（選填，相容舊引擎）

**Step 3｜TypeScript client：`services/valuationClient.ts`**
- `fromXGBoostSnakeCase()` 加入 `shap_factors` → `shapFactors` 對應

**Step 4｜LINE Flex：`services/conversationHandler.ts`**
- `crew2Rows` 加 SHAP 因子一行顯示（`label↑↓%` 格式）

**Step 5｜LIFF 前端：`frontend/src`**
- `types/index.ts`：加 `ShapFactor` interface + `XGBoostValuationResult.shapFactors?`
- `pages/ValuationPage.tsx`：Step 3 結果頁新增 SHAP 橫條圖區塊
