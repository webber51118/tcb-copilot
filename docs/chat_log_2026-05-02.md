## 二、Copilot Studio Agent 設計重構

### 問題診斷

原設計使用「問題」節點（Question Node）收集月收入、總負債等資料，屬於**客服機器人**模式。

**錯誤設計：**
```
觸發程序 → 問題：月收入是多少？→ 問題：總負債是多少？→ 計算 → 回答
```

**正確設計（行員查詢工具）：**
```
行員輸入帶數字的問題 → 直接計算並回答
```

### 解決方案

刪除所有「問題」節點主題，改用生成式 AI + 知識庫。

在「概觀」→「Instructions」設定 Agent 角色與行為規範，讓 AI 直接理解行員問題並回答。

## 三、完整 Instructions 內容

### 角色與核心原則
- 服務對象：合庫銀行個金業務行員（非客戶）
- 行員問什麼直接回答，不反問不必要的問題
- 行員提供數字時立即計算給出結論
- 引用規定標注依據來源
- 使用台灣金融業術語

### DBR 計算規則
- 公式：每月各筆債務還款總額 ÷ 月收入 × 100%
- 上限：≤ 22 倍月收入

### 房貸負債比
- 公式：（所有每月還款 + 18,000）÷ 月收入 ≤ 85%

### 央行受限戶規定（115.3.20 現行版）

| 借款人情況 | 最高成數 | 寬限期 |
|-----------|---------|--------|
| 首購（名下無房貸） | 7成（一般）/ 6成（特定地區） | 最長 3 年 |
| 第2戶（名下已有1筆） | 6成 | ❌ 不得有 |
| 第3戶以上 | 5成 | ❌ 不得有 |
| 高價住宅（台北市7,000萬↑ / 其他6,000萬↑） | 4成 | ❌ 不得有 |
| 公司法人 | 4成 | ❌ 不得有 |
| 餘屋貸款（建商） | 4成 | ❌ 不得有 |

**特定地區**：台北市、新北市、桃園市、台中市、台南市、高雄市、新竹縣、新竹市

### 第二戶換屋協處措施（115.3.20 Q18 新增）

**適用對象**：名下已有1戶房貸，主張實質換屋自住需求，申辦第2戶貸款者

**優惠**：可同時豁免成數限制（6成）＋寬限期禁止

**切結義務**：撥款後 18 個月內出售第1戶並塗銷抵押權

**違約效果**：取消寬限期、收回差額本金、追溯罰息、加收違約金

**重要**：115.3.20 前已撥款之第2戶案件，若原規定較嚴，可比照新規辦理增貸

### 青安貸款正確條件（114.12 版）

| 項目 | 正確值 | 常見錯誤 |
|------|--------|---------|
| 貸款年限 | **最長 40 年** | ~~20年~~ |
| 寬限期 | **最長 5 年** | ~~3年~~ |
| 最高成數 | 8成（不受央行受限戶限制） | — |
| 最高額度 | **1,000 萬元** | ~~350萬（舊版）~~ |

**利率三種計息方式（三選一，不得變更）**

| 方式 | 說明 | 目前利率 |
|------|------|---------|
| 一段式機動 | 基準利率 + 0.555% 全期 | 約 1.775% |
| 二段式機動 | 前2年 + 0.345%，第3年起 + 0.645% | 前2年 1.565% / 後 1.865% |
| 混合式固定 | 前2年固定，第3年起機動 | 第1年 1.745% / 第2年 1.845% / 第3年起 1.865% |

基準利率：郵政2年定儲（113.3.27起為1.72%）
政府補貼：0.375%（至115.7.31，轉貸不適用）

**實施期程**：至 115.7.31，撥款至遲 115.10.31

任一段不得低於：定儲指數月指標利率 + 固定加碼年息 1.55%（浮動計息）

## 四、Demo 測試三組場景

### Demo 1 — 第二戶受限戶（標準）
**輸入**：「客戶名下1筆房貸，台北市買總價1,500萬，可以貸多少？」

**正確回應**：
- 第2戶最高6成 → 1,500萬 × 60% = 900萬
- 無寬限期

### Demo 2 — 第二戶換屋協處（Q18）
**輸入**：「客戶名下1筆房貸，新北市買1,200萬，換屋先買後賣，能貸多少？有寬限期嗎？」

**正確回應**：
- 一般規定：6成 → 720萬，無寬限期
- 換屋協處（Q18切結）：豁免成數限制 + 可有寬限期
- 切結義務：18個月內出售第1戶並塗銷
- 利率下限：定儲指標 + 1.55%

### Demo 3 — 青安貸款
**輸入**：「30歲首購，台北市買1,500萬，想申請青安，可以貸多少？」

**正確回應**：
- 不受央行成數限制（115.3.20 Q4）
- 最高8成 → 最高可貸1,200萬（含青安上限1,000萬）
- 利率三種計息方式（1.565%~1.865%區間）
- 年限最長40年，寬限期最長5年

## 五、知識庫問題診斷與修正

### 問題1：網路搜尋讀到舊資料
**症狀**：Agent 回答第2戶成數為「5成」（第七波舊規定）
**根因**：Copilot Studio 開啟網路搜尋，搜到 115.3.20 前的舊文章

**修正**：
1. 設定 → 生成式 AI → 關閉「搜尋網際網路」
2. Instructions 加入版本說明：「第2戶現行為6成（非5成），以知識庫為準」

### 問題2：青安年限 / 寬限期錯誤
**症狀**：Agent 回答「最長20年，含寬限期3年」
**根因**：規章輯要含多個版本，Agent 讀到舊版條文

**修正**：Instructions 明確標注正確值並說明常見錯誤

### 問題3：知識庫文件過大
**症狀**：回應慢、新舊規定混答
**根因**：
- 政策性規章：4,906 行（含16章，多數與 Demo 無關）
- 授信規章：20,874 行（含31章）
