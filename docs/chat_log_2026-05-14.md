## 四、防詐 Agent 架構確認

### 兩層架構
- **規則引擎**（TypeScript）：`fraudDetector.ts` — 8 項查核，輸出 normal/caution/alert
- **ML 評分**（Python FastAPI, port 8002）：`fraudScoringService.py` — Demo 模式加權評分 / Live 模式 XGBoost + SHAP

### 三級警示路由
- fraud_score ≤ 0.4 → LOW → 行員一鍵確認
- fraud_score 0.4–0.7 → MEDIUM → 資深行員審查
- fraud_score > 0.7 → HIGH → Power Automate → Teams 主管警示

### Demo 場景驗證
趙大山：自由業、文件不符、聯徵 9 次、借款 5 筆 → fraud_score 0.98 HIGH ✅
