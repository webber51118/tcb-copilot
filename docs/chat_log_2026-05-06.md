## 五、Demo 降級詐欺評分修正

**問題**：Python FastAPI（Port 8002）未啟動時，TypeScript fallback 固定回傳 0.5（MEDIUM），無法觸發場景三的 HIGH 風險。

**修正**：`workflowService.ts` fallback 改為完整規則加權計算（對應 Python 邏輯）：

| 特徵 | 貢獻 |
|------|------|
| `documentMatch: false` | +0.35 |
| `creditInquiryCount × 0.06`（上限 0.25）| +0.25 |
| `occupationCode 0 或 4` | +0.15 |
| `livesInBranchCounty: false` | +0.10 |
| `hasSalaryTransfer: false` | +0.08 |
| `existingBankLoans × 0.04`（上限 0.15）| +0.15 |

場景三計算值：0.35 + 0.25 + 0.10 + 0.08 + 0.15 = **0.98**（HIGH ✅）
