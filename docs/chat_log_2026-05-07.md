### 知識庫內容

**商品核心賣點**：可保職業彈性 / 免體檢 / 保費融資 / 身故+完全失能保障

**需求導向三步驟**：
- 步驟一：找出問題（了解家庭結構、主要收入來源）
- 步驟二：擴大問題（萬一發生意外，家人能承擔嗎？）
- 步驟三：解決問題（每月只多 X 元，一個人保全家都有保）

**5 大客群開口話術**：
- 首購族（青安 / 一般房貸）
- 軍公教客戶（國軍輔導 / 軍公教信貸）
- 自營商 / 高風險職業
- 舊貸戶二次行銷
- 對保現場通用話術（3 步驟流程介紹）

**4 個真實理賠案例**：

| 案例 | 理賠金額 | 事故原因 |
|------|---------|---------|
| 廖先生（長庚醫師，37 歲）| 977.2 萬 | 泛舟意外溺斃 |
| 陳女士（台積電工程師，42 歲）| 1,069 萬 | 腦出血完全失能 |
| 蔡小姐（建築師，24 歲）| 372 萬 | 登革熱引發雙下肢截肢 |
| 客戶（企業主，49 歲）| 810 萬 | 新冠病毒呼吸衰竭 |

**6 大反對意見處理**：保險太多 / 保費貴 / 很健康 / 可以保少一點 / 資產夠多 / 提早還款

**合規注意事項**：KYC / 不得搭售 / 借款人=要保人=被保人 / 65 歲規定

## 三、【八】信貸數位化申辦流程 Q&A

### 資料來源

| 文件 | 說明 |
|------|------|
| 信貸作業流程.pdf（113年3月版）| 客戶申請流程 + 營業單位作業流程 |
| 信貸數位化問答集V4.0.pdf | 23 題 Q&A |

### 知識庫內容

**客戶申請 5 步驟**：
1. 信貸申請（官網 / 網銀 / 行動銀行，舊戶 9700 OTP，新戶他行帳戶認證）
2. 進度查詢及補件（前三個月案件，申請書開啟密碼=身分證字號）
3. 修改線上申請文件（需分行 ELOAN 退回後方可修改）
4. 案件准駁及簽約對保（三種選項：不同意/親赴分行/線上對保）
5. 約據下載

**系統自動批覆條件**（三者全符合）：
- 薪轉戶優質客群 OR 任職半年以上員工
- 信用評估 1-8 等（低/中風險）
- 現欠+本筆合計 80 萬以下

**自動審核 6 項目**：9700 / AML / 家事查詢 / 維基 / 代辦人頭戶 / 洗防

**關鍵 Q&A 8 題**：
- 國軍輔導理財不可線上對保（需親赴分行）
- 婉拒後 1 個月內無法再次申請
- 貸放後 2 個月內需 B29+B36 回查
- 100 萬以下免 5450/5470 交易
- 撥款帳戶不含數位存款帳戶；委扣帳戶可含
- 人工核定批覆書需有權人員蓋章（自 112.8.1 起）

## 五、合家保三款商品討論（下午）

### 商品概覽

合庫「合家保」理財專案最熱賣三款：

| 商品 | 幣別 | 繳費期 | 保障期 | 核心賣點 |
|------|------|--------|--------|---------|
| **美好安心** | 美元 | 躉繳 | 7年定期 | 最高折扣2.1%，7年後領滿期金，美元資產配置 |
| **合利超旺** | 台幣 | 2年期 | 終身（至110歲）| 繳2年有終身保障，增值回饋分享金，特定意外加倍理賠 |
| **真美傳家** | 美元 | 6或7年期 | 終身（至111歲）| 最高折扣4.5%，美元終身壽險，二至六級失能豁免保費 |

### 決策方向

**A. productCatalog.ts**：crossSell 新增 `wealthManagement` 欄位（合家保配對商品）

**B. Teams 推薦通知**：行員話術加入房貸壽險 + 合家保話術

### 合家保配對演算法（加權評分）

不用硬編碼 1:1 對應，改用**加權評分演算法**依客戶屬性自動選出最適合的合家保商品：

| 客戶屬性 | 合利超旺（台幣終身）| 美好安心（美元躉繳）| 真美傳家（美元終身）|
|---------|--------------|------------------|---------------|
| 年齡 20-45 | +3 | 0 | +2 |
| 年齡 46-65 | +1 | +3 | +2 |
| 年齡 66+ | 0 | +4 | +1 |
| 軍公教職業 | +3 | +1 | +1 |
| 自營商/高收入 | +1 | +2 | +3 |
| 月收入 < 6 萬 | +3 | 0 | 0 |
| 月收入 6-10 萬 | +2 | +2 | +2 |
| 月收入 > 10 萬 | +1 | +3 | +3 |
| 房貸首購 | +3 | +1 | +1 |
| 以房養老 | 0 | +4 | +2 |
| 優職優利 | +1 | +2 | +4 |

範例：35 歲公務員月收 8 萬 + 青安貸款 → 合利超旺 11 分（最高）→ 推薦合利超旺
