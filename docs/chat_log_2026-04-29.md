## 六、P2：差異化功能

### P2-A：月還款互動試算卡

**月付金公式**（等額攤還 PMT）：
```typescript
monthly = amount * r * (1+r)^n / ((1+r)^n - 1)
// r = annualRate / 12, n = termYears * 12
```

**預設試算利率**：
- 房貸：2.5%
- 信貸：5.5%

**觸發時機**：推薦完成（RECOMMEND 狀態）後自動 push

**互動機制**：

| Quick Reply 按鈕 | 發送文字 | 效果 |
|----------------|---------|------|
| 📅 20年 | `試算:年限:20` | 更新 `session.basicInfo.termYears = 20` |
| 📅 30年 | `試算:年限:30` | 更新 termYears |
| 📅 40年 | `試算:年限:40` | 更新 termYears |
| 💰 -50萬 | `試算:金額:-50` | 金額 -500,000（下限 50 萬） |
| 💰 +50萬 | `試算:金額:+50` | 金額 +500,000（上限房貸 3,000 萬） |
| ✅ 採用此方案 | `繼續` | 進入申請書填寫 |

客戶調整後立即收到新試算卡，不需要重新開始對話。
